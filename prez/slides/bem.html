BEM

  • Naming convention
  • Suffix with __class-name for a child class name
  • Suffix with --modifier-name for a modifier
/****** Button ******/

.button {
    display: inline-block;
    padding: 7px 12px;
    color: #333;
    font: 700 13px/18px Helvetica, arial;
}

/* Button with 'success' modifier */
.button--success {
    color: #4A993E;
}

/* Button with 'danger' modifier */
.button--danger {
    color: #900;
}
/****** Video player ******/

.video-player {
    display: flex;
}

.video-player__list {
    display: flex;
    flex-direction: column;
    max-width: 20rem;
}

.video-player__list__item {
    flex: 1;
}

.video-player__list__item__duration {
    width: 2rem;
    height: auto;
}

BEM

  • (Pro) Naming conventions to avoid conflicts (in theory)
  • (Pro) No complex selector (better performance)
  • (Con) Long class names (big file weight)
