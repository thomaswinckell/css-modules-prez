Conclusion

  • no complicated selector -> performance
  • Minified class names -> css file weight less (BEM: 238kb, CSS modules: 172kb)
  • Scoped class names -> no conflicts
  • Good compatibility with SCSS and Less
  • composes keyword is very limited
  • Hard to integrate with existing framework (keep this in global scope)
  • Webpack needed : don't worth it for small projects that don't need big performance
