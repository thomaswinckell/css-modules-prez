CSS modules

  • Local CSS by default
  • Can switch to global
  • Class composition

CSS modules and Webpack

  • While Webpack is exploring the dependencies, it founds your css file
  • Apply css-loader to convert it to a js module
  • Rename all the classes using a configurable pattern
  • Export all the classes as a JSON object
  • Pattern: [path][name]__[local]--[hash:base64:5]
  • File path : src/TodoList

.root { ... }
.list { ... }
.item { ... }
{
    root : 'src-TodoList__root--3cZQE',
    list : 'src-TodoList__list--3NnME',
    item : 'src-TodoList__item--9xRKZ',
}

CSS modules

  • Switch to global

:global .global-class-name {
    ...
}

CSS modules

  • Composition


.black-color {
    color: #333;
}

.content {
    composes: .black-color from './variables.css';
}
