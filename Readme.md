# css modules presentation

## Run 

Open the index.html on your favorite browser or launch an http-server on the root directory.


## Run demo
```
cd ./examples/css-modules
npm i
npm start
```

## Run demo with production flag
```
cd ./examples/css-modules
npm i
npm run preview
```
## Build demo
```
cd ./examples/css-modules
npm i
npm run build
```
